# Modest Fashion Finder

A web app that aggregates modest clothing from multiple online retailers into one searchable, filterable platform. Built to solve the problem of fragmented, low-quality modest fashion options across the internet.

## Tech Stack
- **Frontend:** React
- **Backend:** Python + FastAPI
- **Database:** PostgreSQL
- **Data Sources:** Web scraping + affiliate APIs

## Status
In active development

## Features 
- Search modest clothing across multiple retailers
- Filter by price, category, size, and design
- Aggregated catalog updated via scrapers and APIs
